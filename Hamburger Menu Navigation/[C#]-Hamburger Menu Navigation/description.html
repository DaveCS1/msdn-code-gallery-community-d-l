Introduction

This project is a simple implementation of Hamburger menu navigation in a UWP app. This is part of my tutorial here : https://code.msdn.microsoft.com/Hamburger-Menu-Navigation-92361623/https://devnextdoor.wordpress.com/2016/04/20/how-to-create-a-simple-hamburger-menu-in-uwp-app-with-navigation

This sample shows you how to create a Hamburger menu using SplitView control and how to load pages into the frame when a particular hamburger menu option is clicked. For a detailed tutorial, do check out the link above.

Concepts

Important Controls in XAML : RelativePanel, SplitView

Building the sample

Open the solution in visual studio 2015 and build it. This project is developed in Windows Universal 10586.0 SDK. The project will compile in any latest Windows 10 SDK. Run the project.
