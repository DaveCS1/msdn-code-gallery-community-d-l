Introduction

Cet exemple montre comment profiter du support de la barre des tâches de Windows 7 et 8 par les MFC, et comment intégrer une application C++/MFC avec la barre des tâches de Windows 7, et utiliser les dernières améliorations de Windows 7 comme l'aperçu dynamique, les raccourcis, la liste des tâches, l'indicateur d'avancement, les pastilles, etc...

Ce code illustre un article du blog MFC www.blogmfc.com : Intégrer une application C++/MFC avec la barre des tâches de Windows 7


Compiler l'exemple (facultatif)

Pour compiler cet exemple, ouvrez le fichier solution (.sln) intitulé MFCTaskbar.sln (Projet C++) avec Visual Studio 2010 SP1 (version professionnelle ou plus). Appuyez sur F7 ou utilisez le menu Build->Build Solution après que le projet soit chargé.

Le Service Pack 1 de Visual Studio 2010 est indispensable.

Executer l'exemple (avec ou sans Visual Studio 2010)

Pour exécuter le programme exemple après l'avoir compilé, appuyez sur F5 (exécuter avec débogage) ou Ctrl-F5 (exécuter sans débogage) depuis Visual Studio 2010 SP1 (version professionnelle ou plus). (Ou sélectionnez l'option correspondante du menu Debug.)
L'archive ZIP contient une version déjà compilée du programme si vous ne disposez pas d'une version professionnelle de Visual Studio 2010.

Description

Description : voir l'article de blog correspondant sur le blog www.blogmfc.com Intégrer une application C++/MFC avec la barre des tâches de Windows 7

Le code et l'article associé montre comment profiter du support de la barre des tâches de Windows 7 et 8 par les MFC, et comment intégrer une application C++/MFC avec la barre des tâches de Windows 7. Cela inclus les fonctionnalités suivantes :

  • Miniatures des fenêtres dans la barre des tâches
  • Gestion des documents récents par Windows
  • Affichage d'un indicateur d'avancement
  • Afficher une pastille de notification
  • Ajouter une liste de tâches
  • Afficher des boutons dans l'aperçu de la fenêtre


Source Code Files

  • MFCTaskbar.cpp MFCTaskbar.h : fichier principal de l'application, gère notamment la liste des tâches et les boutons dans l'aperçu.
  • MainFrm.cpp MainFrm.h : class correspondant à la fenêtre principale de l'application. Gère l'indicateur d'avancement et la pastille de notification
