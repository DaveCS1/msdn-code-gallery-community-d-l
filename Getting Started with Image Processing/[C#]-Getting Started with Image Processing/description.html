Introduction

This sample helps you get started with Image Processing using C# and Windows Forms. This is so that the sample is simple enough for all to understand and so we can focus on the actual image processing code or algorithms.


This sample just gets the ball rolling, implementing basic effects or image processing algorithms. Of course, there are many other effects and if you need any specific one, please do let me know. I would be happy to update the sample with your inputs and if there is sufficient demand, may be migrate this to a WPF version as well :)

Feature Highlights

The major features of this sample include:

  • Invert effect
  • Convert to Gray Scale
  • Control Brightness
  • Control Contrast
  • Control Color

Thus, this sample is a real good start-point to add your own features and continue learning :)

To open a Solution

  1. Extract all files
  2. Start Microsoft Visual Studio
  3. On the File Menu, click Open, and then click Project / Solution
  4. Navigate to the folder containing the ImageProcessing.sln file, select it, and then click Open

To run the Sample

  1. Press F5 to start debugging the application
  2. Open any Image
  3. Click on any Effects button on the toolbar to apply that particular effect
